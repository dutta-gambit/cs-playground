## Project Structure

Each capstone includes:
```
project-name/
├── README.md              # Full requirements doc
├── requirements/          # User stories, use cases
├── diagrams/
│   ├── class-diagram.md   # UML class diagram
│   ├── sequence-diagram.md
│   └── er-diagram.md
├── src/
│   ├── models/
│   ├── services/
│   ├── repositories/
│   └── controllers/
├── tests/
└── docs/
    └── design-decisions.md
```
