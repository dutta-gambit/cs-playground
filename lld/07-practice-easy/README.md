## Problem Template

For each problem, create:
```
problem-name/
├── README.md           # Requirements & approach
├── diagrams/           # Class diagrams, sequence diagrams
├── src/
│   ├── models/         # Entity classes
│   ├── services/       # Business logic
│   └── Main.java       # Demo driver
└── tests/              # Unit tests
```

## How to Approach

1. **Clarify Requirements** (5 min)
2. **Identify Entities** (5 min)
3. **Define Relationships** (5 min)
4. **Draw Class Diagram** (10 min)
5. **Code Core Classes** (20 min)
6. **Handle Edge Cases** (5 min)
